After completing the *IBM Deep Neural Networks with PyTorch* certification on Coursera ([link](https://coursera.org/share/78ba5953c1268a62979a4dbde56ddc64) to certification), I wanted to apply what I learned. 
This is an end to end machine learning project using structural data of numerous protein sequences from [this](https://www.kaggle.com/datasets/shahir/protein-data-set?select=pdb_data_no_dups.csv) Kaggle dataset. 
I begin with an exploratory data analysis (EDA) of the data, then cleaned and transformed the data with normalization, downsampling, and principal component analysis (PCA) to create suitable testing and training datasets. 
From here, I was able to apply what I learned in the *IBM Deep Neural Networks with PyTorch* course, and designed a fully connected neural network with the necessary model training/testing workflow. 
The model's performance is awful (barely breaking 10% testing accuracy), but this project was a good way for me to organize my thoughts and apply what I learned from the Coursera course.
